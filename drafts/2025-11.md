### [Heim G. M., GlobalSearch-rs: A multistart framework for global optimization written in Rust](https://joss.theoj.org/papers/10.21105/joss.09234)
This paper presents `GlobalSearch-rs`, an open-source library written in Rust that implements 
a multistart framework for global optimization of NLP problems.

### [Hovey C. B. and Buche M. R., automesh: Automatic mesh generation in Rust](https://joss.theoj.org/papers/10.21105/joss.08768)
This paper presents `automesh`, an open-source Rust software program that uses a segmentation, typically generated
from a 3D image stack, to create a finite element mesh, composed either of hexahedral
(volumetric) or triangular (isosurface) elements.

### [Scheidwasser N., et al., phylo2vec: a library for vector-based phylogenetic tree manipulation](https://joss.theoj.org/papers/10.21105/joss.09040)
This paper presents the latest release of `phylo2vec`, a high-performance software
package for encoding, manipulating, and analyzing binary phylogenetic trees.
